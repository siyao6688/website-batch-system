## 2. 数据库配置

### 腾讯云MySQL配置

#### 方式一：私有网络（推荐）
1. 在腾讯云控制台创建MySQL数据库
2. 选择私有网络
3. 创建数据库：`website_batch_system`
4. 获取内网地址和连接信息

#### 方式二：公网访问
1. 在腾讯云控制台创建MySQL数据库
2. 获取公网IP和连接信息
3. 配置允许远程连接

#### 配置连接信息

编辑文件：`backend/src/main/resources/application-cloud.yml`

```yaml
spring:
  datasource:
    url: jdbc:mysql://<数据库IP>:3306/website_batch_system?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai&allowPublicKeyRetrieval=true
    username: <数据库用户名>
    password: <数据库密码>
```

### 执行数据库初始化

```bash
# 登录MySQL
mysql -h<数据库IP> -P3306 -uroot -p

# 创建数据库
CREATE DATABASE website_batch_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 授权
GRANT ALL PRIVILEGES ON website_batch_system.* TO 'root'@'%';
FLUSH PRIVILEGES;

# 退出
EXIT;

# 执行schema.sql
mysql -h<数据库IP> -P3306 -uroot -p website_batch_system < backend/src/main/resources/schema.sql
```

## 4. 部署到腾讯云服务器

### 方式一：使用自动部署脚本（推荐新手）

1. **上传脚本到服务器**
```bash
scp deploy-server.sh root@<服务器IP>:/opt/
```

2. **运行脚本**
```bash
ssh root@<服务器IP>
bash /opt/deploy-server.sh
```

3. **后续手动步骤**（见下方）

### 方式二：手动部署

#### 步骤1：上传代码
```bash
# 上传后端代码
scp backend.tar.gz root@<服务器IP>:/opt/

# 上传前端代码
scp frontend/dist.tar.gz root@<服务器IP>:/opt/
```

#### 步骤2：解压代码
```bash
ssh root@<服务器IP>

cd /opt
tar -xzf backend.tar.gz
tar -xzf dist.tar.gz
```

#### 步骤3：配置数据库连接
```bash
nano /opt/website-batch-system/backend/src/main/resources/application-cloud.yml
```
修改数据库连接信息

#### 步骤4：编译后端
```bash
cd /opt/website-batch-system/backend
mvn clean package -DskipTests
```

#### 步骤5：启动后端服务
```bash
systemctl start website-batch-system
systemctl enable website-batch-system
systemctl status website-batch-system
```

#### 步骤6：访问测试
```bash
# 访问后端API
curl http://<服务器IP>:8080/api/excel/companies

# 访问前端
echo "请访问：http://<服务器IP>"
```

## 5. 域名配置

### 1. 购买域名
在腾讯云购买域名（如：example.com）

### 2. 添加DNS解析
在腾讯云DNS控制台添加：
- 类型：A记录
- 主机记录：@
- 记录值：你的服务器公网IP
- TTL：600

### 3. 配置SSL证书（可选但推荐）
使用Let's Encrypt免费证书：

```bash
# 安装certbot
apt install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d <你的域名>

# 自动续期
certbot renew --dry-run
```

## 9. 监控和备份

### 设置自动备份
```bash
# 创建备份脚本
nano /root/backup.sh
```

```bash
#!/bin/bash
BACKUP_DIR="/root/backups"
DATE=$(date +%Y%m%d_%H%M%S)

mkdir -p $BACKUP_DIR

# 备份数据库
mysqldump -h<数据库IP> -P3306 -uroot -p<密码> website_batch_system > $BACKUP_DIR/db_$DATE.sql

# 备份配置文件
cp /opt/website-batch-system/backend/src/main/resources/application-cloud.yml $BACKUP_DIR/config_$DATE.yml

# 删除7天前的备份
find $BACKUP_DIR -type f -mtime +7 -delete

echo "备份完成：$DATE"
```

```bash
# 添加到cron
crontab -e
# 添加以下行（每天凌晨2点执行）
0 2 * * * /root/backup.sh
```
