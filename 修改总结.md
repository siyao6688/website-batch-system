## 🎯 核心功能

### Excel导入流程

1. **下载模板**
   - 点击"下载模板"按钮
   - 下载CSV格式模板（可直接在Excel中打开）
   - 模板包含5列：序号、公司名称、邮箱、域名、备案号

2. **填写数据**
   - 按照模板格式填写公司信息
   - 序号从1开始递增
   - 公司名称必填
   - 域名必填且唯一

3. **上传文件**
   - 点击"上传Excel文件"按钮
   - 选择填写好的Excel/CSV文件
   - 系统自动解析并导入数据

4. **查看结果**
   - 自动刷新公司列表
   - 显示导入成功的公司
   - 可分页查看

### 数据存储

- **数据库**：腾讯云MySQL数据库
- **表**：companies
- **字段**：
  - id - 主键ID
  - serial_number - 序号
  - company_name - 公司名称
  - email - 邮箱
  - domain - 域名
  - icp_number - 备案号
  - created_at - 创建时间
  - updated_at - 更新时间

## 📋 数据库初始化

### 方式一：使用schema.sql（推荐）

```bash
# 登录MySQL
mysql -h<数据库IP> -P3306 -uroot -p

# 创建数据库
CREATE DATABASE website_batch_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 授权
GRANT ALL PRIVILEGES ON website_batch_system.* TO 'root'@'%';
FLUSH PRIVILEGES;

# 执行schema
SOURCE /path/to/schema.sql;
```

### 方式二：使用Spring自动创建

Spring Boot的JPA会自动根据实体类创建表（如果配置了ddl-auto: update）

**配置文件**：`application-cloud.yml`

```yaml
spring:
  jpa:
    hibernate:
      ddl-auto: update  # 自动创建/更新表结构
    show-sql: true      # 显示SQL语句
```

## ⚠️ 注意事项

1. **域名唯一性**
   - 每个域名在数据库中必须唯一
   - 导入时会检查域名是否已存在
   - 如果域名重复，导入会失败

2. **数据验证**
   - 公司名称不能为空
   - 域名不能为空且必须唯一
   - 邮箱格式建议验证（前端可添加）

3. **编码问题**
   - 确保使用UTF-8编码
   - CSV文件开头添加 BOM：`\uFEFF`

4. **序号字段**
   - 序号是Excel中的行号
   - 不需要手动填写，系统会自动设置

## 🛠️ 常见问题

### Q1: Excel导入失败怎么办？
**A:**
1. 检查模板格式是否正确
2. 查看后端日志：`tail -f /var/log/website-batch-system/*.log`
3. 确保数据库连接正常

### Q2: 数据库连接失败？
**A:**
1. 检查腾讯云数据库IP和端口
2. 确认数据库用户名和密码
3. 检查防火墙和安全组设置
4. 验证网络连接：`telnet <数据库IP> 3306`

### Q3: 如何修改字段？
**A:**
1. 修改Company.java实体类
2. 修改ExcelParser.java解析器
3. 修改schema.sql表结构
4. 运行数据库迁移脚本

### Q4: 如何备份数据？
**A:**
```bash
# 导出数据库
mysqldump -h<数据库IP> -P3306 -uroot -p website_batch_system > backup.sql

# 恢复数据库
mysql -h<数据库IP> -P3306 -uroot -p website_batch_system < backup.sql
```
