## 🚀 快速开始

### 方式一：本地测试（推荐先测试）

#### Windows系统
```bash
# 1. 启动后端
cd backend
mvn clean package -DskipTests
java -jar target/website-batch-system-1.0.0.jar

# 2. 启动前端（新终端）
cd frontend
npm install
npm run dev

# 3. 访问
# http://localhost:5173/excel
```

#### Linux/Mac系统
```bash
# 1. 启动所有服务
bash start.sh

# 2. 访问
# http://localhost:5173/excel
```

### 方式二：部署到腾讯云服务器

#### 方式A：使用自动部署脚本（推荐新手）

1. **上传脚本到服务器**
```bash
scp deploy-server.sh root@<服务器IP>:/opt/
```

2. **运行脚本**
```bash
ssh root@<服务器IP>
bash /opt/deploy-server.sh
```

3. **完成后续步骤**（见下文）

#### 方式B：手动部署

1. **阅读快速配置指南**
   ```bash
   cat 快速配置指南.md
   ```

2. **按照指南步骤部署**

## 🔍 故障排查

### 问题1：脚本无法运行

**原因**：权限不足

**解决**：
```bash
# Linux/Mac
chmod +x deploy-server.sh start.sh start.bat
```

### 问题2：Maven编译失败

**原因**：依赖下载慢或网络问题

**解决**：
```bash
# 配置阿里云Maven镜像
# 编辑 ~/.m2/settings.xml
```

### 问题3：后端启动失败

**解决**：
```bash
# 查看详细日志
tail -f backend.log

# 检查端口占用
netstat -tlnp | grep 8080
```

### 问题4：前端无法访问

**解决**：
```bash
# 检查Nginx状态
systemctl status nginx

# 查看Nginx日志
tail -f /var/log/nginx/website-batch-system-error.log
```
