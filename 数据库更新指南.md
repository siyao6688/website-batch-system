## 🔄 更新数据库的方法

### 方法一：删除旧数据库重新创建（推荐）

```bash
# 1. 删除旧数据库
mysql -u root -p -e "DROP DATABASE IF EXISTS website_batch_system;"

# 2. 创建新数据库并执行schema
mysql -u root -p -e "CREATE DATABASE website_batch_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 3. 执行表结构
mysql -u root -p website_batch_system < backend/src/main/resources/schema.sql
```

---

### 方法二：使用 Docker Compose 重建（推荐）

```bash
# 1. 停止并删除旧容器和数据
docker-compose down -v

# 2. 重新启动（会自动执行schema.sql）
docker-compose up -d

# 3. 查看日志确认
docker-compose logs mysql
```

## 📊 数据库表结构

### companies 表
```
companies (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  serial_number INT,                          -- 序号
  company_name VARCHAR(200) NOT NULL,         -- 公司名称
  email VARCHAR(100),                         -- 邮箱
  domain VARCHAR(100) NOT NULL UNIQUE,        -- 域名
  icp_number VARCHAR(50),                     -- 备案号
  has_website BOOLEAN DEFAULT FALSE,          -- 是否搭建官网
  template_id BIGINT,                         -- 模板ID
  is_published BOOLEAN DEFAULT FALSE,         -- 是否发布
  is_active BOOLEAN DEFAULT TRUE,             -- 是否激活
  publish_date DATETIME,                      -- 发布时间
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)
```

### website_contents 表
```
website_contents (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  company_id BIGINT NOT NULL,                 -- 公司ID
  company_name VARCHAR(200) NOT NULL,         -- 公司名称
  content_type VARCHAR(50) NOT NULL,          -- 内容类型
  title VARCHAR(200),                         -- 标题
  description TEXT,                           -- 描述
  content_detail TEXT,                        -- 内容详情
  image_url VARCHAR(500),                     -- 图片URL
  sort_order INT DEFAULT 0,                   -- 排序
  is_active BOOLEAN DEFAULT TRUE,             -- 是否启用
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)
```

### website_templates 表
```
website_templates (
  id BIGINT AUTO_INCREMENT PRIMARY KEY,
  template_name VARCHAR(100) NOT NULL,        -- 模板名称
  template_code VARCHAR(50) NOT NULL UNIQUE,  -- 模板代码
  preview_image VARCHAR(500),                 -- 预览图
  description TEXT,                           -- 描述
  is_active BOOLEAN DEFAULT TRUE,             -- 是否启用
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP
)
```
