## 📋 功能说明

### 1. 数据概览

顶部显示 4 个统计卡片：
- **公司总数** - 显示所有公司数量
- **已发布** - 已发布的公司数量
- **正常状态** - 激活状态的公司数量
- **已生成网站** - 已生成网站的公司数量

### 2. 搜索功能

在顶部搜索框输入：
- 公司名称
- 域名
- 备案号

实时过滤显示匹配结果。

### 3. 公司列表

主要功能区域：
- 显示公司详细信息
- 状态标签（正常/停用/已发布/草稿）
- 网站状态（已生成/未生成）
- 操作按钮

### 4. 操作功能

**编辑**：
- 点击编辑图标
- 修改公司信息
- 保存更新

**生成网站**：
- 点击生成图标
- 选择模板
- 选择是否立即发布

**发布**：
- 点击确认图标
- 需要先生成网站
- 发布后网站立即可访问

**取消发布**：
- 点击时间图标
- 网站立即下线

**删除**：
- 点击删除图标
- 确认删除操作
- 删除后无法恢复

## 🎯 使用流程

### 新增公司流程

1. 点击顶部 **"新增"** 按钮
2. 填写公司信息：
   - 公司名称（必填）
   - 邮箱（选填）
   - 域名（必填）
   - 备案号（选填）
3. 设置状态：
   - 是否激活
   - 是否搭建官网
4. 点击 **"提交"** 保存

### 导入公司流程

1. 访问 `http://localhost:5173/excel`
2. 下载模板
3. 填写数据
4. 上传文件
5. 返回管理页面查看

### 生成网站流程

1. 在公司列表中找到公司
2. 点击 **"生成网站"** 图标
3. 选择模板（可选）
4. 选择是否立即发布
5. 点击 **"生成并发布"**

## 🔧 自定义配置

### 修改 API 地址

编辑 `.env` 文件：
```bash
VITE_API_BASE_URL=http://your-server:8080/api
```

### 修改主题色

在 `AdminDashboard.css` 中修改：
```css
.ant-btn-primary {
  background: linear-gradient(135deg, #1890ff 0%, #096dd9 100%);
}
```

### 修改统计指标

在 `AdminDashboard.jsx` 中修改：
```jsx
const updateStats = (data) => {
  setStats({
    total: data.length,
    published: data.filter(c => c.isPublished).length,
    active: data.filter(c => c.isActive).length,
    hasWebsite: data.filter(c => c.hasWebsite).length
  });
};
```
