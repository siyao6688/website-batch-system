# 网站批量生成系统 - 启动指南

## 环境要求

- Java 17+
- Node.js 16+
- MySQL 8.0+

## 快速启动步骤

### 方式1: Docker Compose（推荐）

```bash
# 一键启动所有服务
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

访问地址：
- 后端API: http://localhost:8080/api
- 前端管理: http://localhost:5173

### 方式2: 本地启动

#### 1. 数据库配置

```bash
# 登录MySQL
mysql -u root -p

# 创建数据库
CREATE DATABASE website_batch_system CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

# 导入表结构
source /path/to/backend/src/main/resources/schema.sql;

# 退出
exit;
```

#### 2. 后端启动

```bash
cd backend

# 修改配置文件中的数据库密码
vi src/main/resources/application-dev.yml

# 启动后端
mvn spring-boot:run
```

后端启动后访问: http://localhost:8080/api

#### 3. 前端启动

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

前端启动后访问: http://localhost:5173

## Excel导入流程

1. 访问 http://localhost:5173
2. 点击左侧菜单 "Excel导入"
3. 下载Excel模板
4. 填写公司信息
5. 上传Excel文件
6. 系统自动解析并导入公司信息

## 使用流程

### 1. 导入公司信息

- 使用Excel导入功能批量添加公司信息
- 必填字段：公司名称、域名

### 2. 编辑公司信息

- 在公司列表中点击"编辑"
- 填写详细的公司信息（包括备案信息、腾讯云信息等）
- 选择一个模板

### 3. 发布网站

- 在公司列表中点击"发布"
- 系统会自动生成公司网站
- 生成的网站保存在服务器的网站输出目录

### 4. 访问网站

生成的网站可以通过域名直接访问，例如：
- http://hfojinznd.cn

## 配置说明

### 后端配置文件

文件位置: `backend/src/main/resources/application-dev.yml`

主要配置项：
- 数据库连接信息
- 网站输出路径: `website.output-path`
- 模板路径: `website.templates-path`

### 前端配置文件

文件位置: `frontend/.env`

主要配置项：
- API地址: `VITE_API_BASE_URL`

## 常见问题

### 1. 数据库连接失败

检查配置文件中的数据库连接信息是否正确。

### 2. Excel导入失败

确保Excel文件格式正确，必须包含表头。

### 3. 网站生成失败

检查模板文件是否存在，检查输出路径是否有写入权限。

### 4. 前端无法连接后端

确保后端服务已启动，检查API地址配置是否正确。
